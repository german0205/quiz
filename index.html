Conectar base de datos

Para votación en tiempo real con múltiples dispositivos necesitás una cuenta gratuita en Supabase. Completá los datos una sola vez — se guardan en este navegador.

Supabase Project URL
Supabase Anon Key
Cómo obtener los datos:
1. Creá cuenta en supabase.com (gratis)
2. Nuevo proyecto → Settings → API
3. Copiá Project URL y anon public key
4. En el SQL Editor ejecutá:
CREATE TABLE IF NOT EXISTS quiz_votes (
  id BIGSERIAL PRIMARY KEY,
  session_id TEXT NOT NULL,
  q_index INTEGER NOT NULL,
  option_index INTEGER NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);
CREATE TABLE IF NOT EXISTS quiz_state (
  session_id TEXT PRIMARY KEY,
  current_q INTEGER DEFAULT 0,
  timer_end BIGINT DEFAULT 0,
  revealed BOOLEAN DEFAULT FALSE,
  active BOOLEAN DEFAULT TRUE,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE quiz_votes ENABLE ROW LEVEL SECURITY;
ALTER TABLE quiz_state ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all_votes" ON quiz_votes FOR ALL USING (true) WITH CHECK (true);
CREATE POLICY "allow_all_state" ON quiz_state FOR ALL USING (true) WITH CHECK (true); 5. Activá Realtime para quiz_state y quiz_votes en Database → Replication
